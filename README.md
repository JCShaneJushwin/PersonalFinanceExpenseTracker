An interactive Power BI dashboard built to analyze and visualize personal financial data across multiple years. This project delivers clear insights into income, expenses, savings, and spending behavior using dynamic KPIs and intuitive visual storytelling.
Key Features:
⭐ KPIs for Total Income, Total Expense, Total Savings, Expense %, and Savings %
⭐ Year-wise comparison of income, expenses, and savings
⭐ Category-wise expense analysis (rent, groceries, EMI, health, leisure, shopping)
⭐ Savings distribution across different investment types
⭐ Interactive slicers for year, component, and type filtering
⭐ Detailed tabular view for multi-year financial breakdown

Tools & Skills Used:

1. Power BI

2. Data Cleaning & Transformation

3. DAX Measures

4. Data Visualization & Dashboard Design
